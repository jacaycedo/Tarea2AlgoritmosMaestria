Explicación implementación:

El proyecto recibe por parametros dos rutas, la primera corresponde al archivo de entrada y el segundo al archivo de salida.

El archivo de entrada tienen que tener el siguiente formato
	- En la primera linea debe estar el numero de trabajadores y de trabajos.
	- En las lineas posteriores deben haber tuplas de trabajador \t trabajos, en donde esto representa que el trabajador puede realizar cierto trabajo.
	- Debe ser un archivo separado por tabs.

Hay problemas de ejemplo con sus respectivos diagramas en la carpeta Datos.

Para este proyecto se requirieron 2 clases:

	- ejecutable.java: Contiene el codigo general y la interacción con la consola de los distintos casos.
	- solucion.java: Contiene la implementación de el algoritmo de Edmonds-Karp, aplicado al caso de los trabajadores.
